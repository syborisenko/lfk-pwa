# ЛФК PWA — Персональный трекер терапевтических упражнений

## О проекте

PWA-приложение для отслеживания ЛФК тренировок, составленных реабилитологом для конкретного пациента (Сергей). Single-file архитектура: один `index.html` (~170KB) с инлайн CSS и JS, без фреймворков, без бандлера. Деплой на GitHub Pages.

- Репо: https://github.com/syborisenko/lfk-pwa
- Деплой: https://syborisenko.github.io/lfk-pwa/

## Здоровье пациента

**Основные проблемы:**
- Перекос таза (пельвик тилт) — слабые ягодицы, особенно средние
- Боль в пояснице — спазм грушевидной мышцы, возможно сдавливание седалищного нерва
- Передний наклон головы ("голова черепахи") — от сидячей работы
- Грудной кифоз (сутулость) — укороченные грудные, слабые ромбовидные
- Круглые плечи — слабая задняя цепь плечевого пояса
- Слабый кор — нестабильность поясницы

**Оборудование:** резинки, фоам-роллер, йога-блоки, гантели, мяч/подушка, фитбол.

## Архитектура

- `index.html` — всё приложение (CSS + HTML + JS)
- `manifest.json` — PWA манифест
- `sw.js` — Service Worker (Network First с кэш-фоллбэком)
- `icon-192.png`, `icon-512.png` — иконки
- `.github/workflows/deploy.yml` — автодеплой на GitHub Pages

### Данные
- localStorage ключ: `lfk_v6`
- Миграция с v5/v4 автоматическая
- ~104 упражнения в массиве `EX`
- Слоты по дням в массиве `SLOTS` с полем `phase`

## Структура тренировок

3 дня, строгий реабилитационный порядок (шафл меняет упражнения внутри слотов, но порядок слотов фиксирован):

```
Разминка → Активация → Стабилизация → Укрепление → Растяжка → Релиз → Расслабление
```

- **День A** — Таз, поясница, ноги (12 слотов, ~32-35 мин)
- **День B** — Шея, осанка, плечи (16 слотов, ~38-42 мин)
- **День C** — Кор, стабилизация (11 слотов, ~33-35 мин)

## Ключевые фичи

- **Авто-плеер** — полноэкранный режим тренировки: автотаймер, отдых 15 сек между подходами таймерных упражнений, подготовка 10 сек перед таймерными, SVG-кольцо прогресса с плавной анимацией, вибрация, wake lock (включая iOS через silent video)
- **Для упражнений на повторы** — без автоматических таймеров/отдыха, только кнопка "Готово"
- **Прогрессия нагрузки** — +/- кнопки на карточках, мягкие подсказки через 2 недели
- **Отключение упражнений** — toggle в библиотеке и ✕ на карточке при травмах
- **Статистика** — календарь тренировок + графики (бары по неделям, пай-чарт баланса дней)
- **Экспорт/импорт** — JSON файл через настройки
- **Мягкие напоминания** — баннер на главной с рекомендацией следующего дня

## Стилевые соглашения

- Весь код в одном файле — не разбивать
- `var` вместо `let/const` (совместимость)
- Функции через `function declarations`
- CSS минифицирован в `<style>`
- Кириллица в строках (интерфейс на русском)
- Без внешних зависимостей и библиотек

## Планы на будущее

- Трекинг самочувствия: ежедневный опрос и обратная связь после тренировки
- Корректировка программы в зависимости от ощущений (боль, усталость, прогресс)
- Возможно: адаптивная сложность на основе данных самочувствия
